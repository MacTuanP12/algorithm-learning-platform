# algorithm-learning-platform
# Algorithm Visualizer

Interactive algorithm visualization platform.

## Features

- Sorting Algorithms Visualization
- Graph Algorithms Visualization
- Dynamic Programming Visualizer
- Step-by-step execution

## Tech Stack

Backend
- Java
- Spring Boot

Frontend
- React
- Vite
- D3.js

## Algorithms

Sorting
- Bubble Sort
- Quick Sort
- Merge Sort

Searching
- Binary Search
- Linear Search

Graph
- BFS
- DFS
- Dijkstra

## Architecture

React UI → Spring Boot API → Algorithm Engine

## Run

Backend

cd backend
mvn spring-boot:run

Frontend

cd frontend
npm install
npm run dev
